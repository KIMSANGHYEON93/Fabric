## 소개

완전한 100 경험치

- 3분

Fabric 데이터 엔지니어 또는 개발자는 데이터를 저장, 처리 및 분석하는 Fabric 항목을 설계, 구축 및 유지 관리하는 업무를 담당합니다. 팀과 협력하여 개발 단계부터 운영 단계까지 다양한 Fabric 작업 공간에서 변경 사항을 배포합니다. 여러 엔지니어가 프로젝트에 참여하기 때문에 Fabric 항목 변경 사항을 점진적으로 배포 및 통합하고, 테스트 및 신속한 배포가 가능한 배포 프로세스를 거쳐야 합니다. Fabric의 수명 주기 관리를 통해 이러한 작업을 수행할 수 있습니다.

Fabric의 수명 주기 관리는 배포 파이프라인과 소스 제어 통합을 활용하여 지속적인 통합 및 지속적인 배포(CI/CD)를 지원합니다. Fabric을 GitHub 및 Azure DevOps와 같은 버전 제어 시스템과 통합하여 배포 프로세스를 개선하고 자동화된 테스트를 통합할 수 있습니다.

이 모듈을 마치면 Fabric에서 소스 제어를 사용하는 방법, Fabric 배포 파이프라인을 사용하는 방법, Fabric REST API를 CI/CD 자동화에 사용하는 방법을 알게 됩니다.
