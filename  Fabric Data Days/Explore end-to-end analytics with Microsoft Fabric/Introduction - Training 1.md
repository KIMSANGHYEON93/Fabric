## 소개

완전한 100 경험치

- 1분

Microsoft Fabric은 데이터 전문가와 기업이 데이터 프로젝트를 위해 협업할 수 있는 단일 통합 환경을 제공하는 엔드투엔드 분석 플랫폼입니다. Fabric은 단일 환경에서 데이터를 수집, 저장, 처리 및 분석할 수 있는 통합 서비스 세트를 제공합니다.

Microsoft Fabric은 일반 시민과 전문 데이터 실무자 모두를 위한 도구를 제공하며, 기업의 의사 결정에 필요한 도구와 통합됩니다. Fabric에는 다음과 같은 서비스가 포함됩니다.

- 데이터 엔지니어링
- 데이터 통합
- 데이터웨어하우징
- 실시간 인텔리전스
- 데이터 과학
- 비즈니스 인텔리전스

또한 Microsoft Fabric은 데이터 전문가와 비즈니스 사용자 모두에게 지능형 코드 완성, 자연어에서 SQL로의 변환, 자동화된 통찰력, 상황에 맞는 지원을 제공하여 모든 작업 부하에서 생산성을 향상시키는 생성적 AI 어시스턴트인 Copilot을 통합합니다.

이 모듈에서는 Fabric 플랫폼을 소개하고, Fabric의 대상에 대해 논의하고, Fabric 서비스를 살펴보고, Copilot이 분석 경험을 어떻게 향상시키는지 살펴봅니다.
