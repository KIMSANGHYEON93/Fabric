## 소개

완전한 100 경험치

- 1분

Microsoft Fabric의 기반은 **OneLake** 확장 가능 스토리지 계층 위에 구축된 **레이크하우스로**, 빅데이터 처리를 위해 Apache Spark와 SQL 컴퓨팅 엔진을 사용합니다. 레이크하우스는 다음을 결합한 통합 플랫폼입니다.

- *데이터 레이크* 의 유연하고 확장 가능한 스토리지
- *데이터웨어 하우스* 의 데이터를 쿼리하고 분석하는 능력

회사에서 주문 내역, 재고 수준, 고객 정보와 같은 거래 시스템의 정형 데이터를 저장하기 위해 데이터 웨어하우스를 사용하고 있다고 가정해 보겠습니다. 기존 데이터 웨어하우스 인프라로는 관리 및 분석이 어려운 소셜 미디어, 웹사이트 로그, 외부 소스 등에서 비정형 데이터를 수집합니다. 회사의 새로운 목표는 여러 소스에서 다양한 형식의 데이터를 분석하여 의사 결정 역량을 향상시키는 것이었고, 이에 따라 Microsoft Fabric을 선택했습니다.

이 모듈에서는 Microsoft Fabric의 레이크하우스가 SQL을 사용하여 쿼리할 수 있는 파일과 테이블에 대한 확장 가능하고 유연한 데이터 저장소를 제공하는 방법을 살펴봅니다.
