마이크로소프트 패브릭

레이크하우스는 데이터 레이크 스토리지의 유연성과 데이터웨어하우스 분석을 결합합니다. Microsoft Fabric은 단일 SaaS 플랫폼에서 포괄적인 분석을 위한 레이크하우스 솔루션을 제공합니다.

## 학습 목표

이 모듈에서는 다음 내용을 학습합니다.

- Microsoft Fabric의 레이크하우스의 핵심 기능과 성능을 설명합니다.
- 호숫가 주택을 만들어 보세요.
- 레이크하우스의 파일과 테이블에 데이터를 수집합니다.
- SQL을 사용하여 레이크하우스 테이블을 쿼리합니다.
