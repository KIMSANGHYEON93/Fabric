레이크하우스의 여러 층을 별도의 작업 공간에 보관하는 이점은 무엇입니까?

이를 통해 보안을 강화하고, 용량 사용을 관리하고, 비용 효율성을 최적화할 수 있습니다.

레이크하우스의 여러 층을 별도의 작업 공간에 보관하는 것에는 아무런 이점이 없습니다.
