## 소개

완전한 100 경험치

- 2분

데이터 중심 시대에 방대하고 다양한 데이터 소스를 활용하는 것은 비즈니스 성공에 매우 중요합니다. 데이터 레이크와 데이터 웨어하우스를 결합한 Fabric 레이크하우스는 이러한 데이터를 관리하고 분석하는 데 이상적인 플랫폼을 제공합니다. 메달리온 아키텍처는 레이크하우스 기반 분석의 업계 표준으로 자리 잡았습니다.

메달리온 아키텍처는 레이크하우스 환경에 구조와 효율성을 더합니다. 이 모듈은 청동, 실버, 골드 레이어를 안내하여 데이터를 효과적으로 구성, 정제 및 큐레이션하는 방법을 알려줍니다.

숙련된 데이터 엔지니어든 분석 초보자든, 이 모듈은 실제 사례와 실습을 통해 모든 수준의 경험에 부합합니다. 시작하기 전에 Fabric 데이터 레이크하우스에 익숙하고 SQL 및 Power BI에 대한 기본적인 이해가 필요합니다.

이 모듈에서는 Fabric 레이크하우스를 위한 메달리온 아키텍처를 탐색하고 구축하고, Fabric 레이크하우스의 데이터를 쿼리하고 보고하고, Fabric 레이크하우스의 보안 및 거버넌스를 위한 모범 사례를 설명합니다.
