Microsoft Fabric에서 메달리온 아키텍처 설계의 잠재력을 살펴보세요. 레이크하우스의 브론즈, 실버, 골드 계층에서 데이터를 구성하고 변환하여 최적화된 분석을 구현하세요.

## 학습 목표

이 모듈에서는 다음 내용을 학습합니다.

- 데이터 관리에서 메달리온 아키텍처를 사용하는 원칙을 설명하세요.
- Microsoft Fabric 환경 내에서 메달리온 아키텍처 프레임워크를 적용합니다.
- Power BI에서 DirectLake를 사용하여 레이크하우스에 저장된 데이터를 분석합니다.
- 메달리온 아키텍처에 저장된 데이터의 보안과 거버넌스를 보장하기 위한 모범 사례를 설명합니다.
