## Spark 코드 실행

완전한 100 경험치

- 3분

*Microsoft Fabric에서 Spark 코드를 편집하고 실행하려면 노트북을* 사용 하거나 *Spark 작업을* 정의할 수 있습니다.

## 노트북

Spark를 사용하여 데이터를 대화형으로 탐색하고 분석하려면 노트북을 사용하세요. 노트북을 사용하면 여러 언어로 작성된 텍스트, 이미지, 코드를 결합하여 다른 사람과 공유하고 협업할 수 있는 대화형 항목을 만들 수 있습니다.

![Microsoft Fabric의 노트북 스크린샷.](https://learn.microsoft.com/en-us/training/wwl/use-apache-spark-work-files-lakehouse/media/notebook.png)

노트북은 하나 이상의 *셀로* 구성되며, 각 셀에는 마크다운 형식의 콘텐츠나 실행 코드가 포함될 수 있습니다. 노트북에서 코드를 대화형으로 실행하고 결과를 즉시 확인할 수 있습니다.

## Spark 작업 정의

자동화된 프로세스의 일부로 Spark를 사용하여 데이터를 수집하고 변환하려는 경우, Spark 작업을 정의하여 주문형 또는 일정에 따라 스크립트를 실행할 수 있습니다.

![Microsoft Fabric의 Spark 작업 정의 스크린샷.](https://learn.microsoft.com/en-us/training/wwl/use-apache-spark-work-files-lakehouse/media/spark-job.png)

Spark 작업을 구성하려면 작업 공간에 Spark 작업 정의를 생성하고 실행할 스크립트를 지정합니다. 참조 파일(예: 스크립트에서 사용되는 함수 정의가 포함된 Python 코드 파일)과 스크립트가 처리하는 데이터가 포함된 특정 레이크하우스에 대한 참조를 지정할 수도 있습니다.
