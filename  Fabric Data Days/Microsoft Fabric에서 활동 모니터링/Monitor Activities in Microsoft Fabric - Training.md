모니터링을 통해 데이터 시스템 상태를 파악할 수 있습니다. Microsoft Fabric의 Monitoring Hub는 Fabric 활동 데이터를 수집하고 집계합니다. Microsoft Fabric Activator는 스트리밍 데이터에서 패턴이나 조건이 감지될 때 조치를 취할 수 있도록 도와줍니다.

## 학습 목표

이 모듈에서는 다음 내용을 학습합니다.

- Microsoft Fabric에 모니터링 개념 적용
- Microsoft Fabric에서 모니터링 허브 사용
- Microsoft Fabric에서 Activator를 사용하여 작업 트리거

[시작](https://learn.microsoft.com/en-us/training/modules/monitor-fabric-items/1-introduction/) 추가하다
