## 소개

완전한 100 경험치

- 2분

*실시간 인텔리전스에서 이벤트스트림을* 사용하면 코드를 작성하지 않고도 실시간 이벤트를 캡처, 변환 및 라우팅할 수 있습니다. 이벤트스트림에서 이벤트 소스, 대상 및 변환을 설정하여 데이터를 수집하고, 필요에 따라 처리한 후 대상으로 라우팅할 수 있습니다.

Fabric에서 Eventstream을 사용하는 주요 이점은 다음과 같습니다.

- 데이터를 캡처, 변환하고 다양한 목적지로 라우팅할 수 있습니다. 드래그 앤 드롭 기능을 사용하면 코드를 작성하지 않고도 이벤트 데이터 처리 로직을 만들 수 있습니다.
- Azure Event Hubs, Azure IoT Hub, Azure Storage, Apache Kafka 등 다양한 소스에서 이벤트 데이터를 수집하기 위한 여러 소스 커넥터가 있습니다.
- 저장, 추가 분석 또는 처리를 위해 다양한 목적지로 데이터를 보낼 수 있습니다.

이 모듈에서는 이벤트 스트림을 생성하고 이를 사용하여 이동 중인 데이터를 수집하고 변환하고, 추가 처리 및 분석을 위해 대상지로 라우팅하는 방법을 알아봅니다.
