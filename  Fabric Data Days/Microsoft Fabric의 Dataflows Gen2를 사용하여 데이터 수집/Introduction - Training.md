## 소개

완전한 100 경험치

- 2분

Microsoft Fabric은 데이터 엔지니어링, 통합 및 분석을 위한 통합 솔루션을 제공합니다. 엔드투엔드 분석의 핵심 단계는 데이터 수집입니다. 데이터 흐름 Gen2는 여러 소스에서 데이터를 수집하고 변환한 후, 정제된 데이터를 다른 목적지로 전송하는 데 사용됩니다. 데이터 흐름 Gen2는 더욱 복잡한 활동 오케스트레이션을 위해 데이터 파이프라인에 통합될 수 있으며, Power BI에서 데이터 소스로도 사용할 수 있습니다.

전 세계에 매장을 둔 소매 회사에서 일한다고 가정해 보겠습니다. 데이터 엔지니어로서 다양한 소스의 데이터를 데이터 분석 및 보고에 적합한 형식으로 준비하고 변환해야 합니다. 회사는 여러 매장의 분산된 데이터 소스를 통합하는 시맨틱 모델을 요구합니다. Dataflows Gen2를 사용하면 일관성을 보장하도록 데이터를 준비하고 원하는 위치에 데이터를 스테이징할 수 있습니다. 또한 데이터 재사용을 지원하고 손쉽게 업데이트할 수 있습니다. 데이터플로가 없다면 모든 소스에서 데이터를 수동으로 추출하고 변환해야 하므로 시간이 많이 소요되고 오류가 발생하기 쉽습니다.

이 모듈에서는 Microsoft Fabric에서 Dataflows Gen2를 사용하여 데이터 수집 요구 사항을 충족하는 방법을 설명합니다.
