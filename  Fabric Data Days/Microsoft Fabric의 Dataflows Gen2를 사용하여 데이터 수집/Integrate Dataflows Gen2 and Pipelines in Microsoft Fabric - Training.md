## Microsoft Fabric에서 Dataflows Gen2 및 파이프라인 통합

완전한 100 경험치

- 2분

Dataflows Gen2는 Microsoft Fabric에서 데이터 변환을 위한 탁월한 옵션을 제공합니다. 데이터 흐름과 파이프라인을 함께 사용하면 변환된 데이터에 추가 작업을 수행해야 할 때 유용합니다.

데이터 파이프라인은 데이터 엔지니어링에서 흔히 사용되는 개념이며, 다양한 활동을 조율할 수 있도록 지원합니다. 일반적인 활동은 다음과 같습니다.

- 데이터 복사
- 데이터 흐름 통합
- 노트북 추가
- 메타데이터 가져오기
- 스크립트 또는 저장 프로시저 실행

파이프라인은 특정 순서대로 작업을 완료하는 시각적인 방법을 제공합니다. 데이터 흐름을 사용하여 데이터를 수집, 변환하고 Fabric 데이터 저장소에 저장할 수 있습니다. 그런 다음 데이터 흐름을 파이프라인에 통합하여 데이터 흐름이 완료된 후 스크립트나 저장 프로시저를 실행하는 등의 추가 작업을 조정할 수 있습니다.

파이프라인은 트리거를 통해 예약하거나 활성화하여 데이터 흐름을 실행할 수도 있습니다. 파이프라인을 사용하여 데이터 흐름을 실행하면 데이터 흐름을 수동으로 실행할 필요 없이 필요할 때 데이터를 새로 고칠 수 있습니다. 엔터프라이즈급 데이터나 자주 변경되는 데이터를 처리하는 경우, 자동화를 통해 다른 업무에 집중할 수 있습니다.
