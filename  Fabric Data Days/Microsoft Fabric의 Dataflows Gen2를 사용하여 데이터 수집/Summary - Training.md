## 요약

완전한 100 경험치

- 2분

이 모듈에서는 두 데이터 엔지니어가 데이터를 수집, 변환하고 레이크하우스와 같은 Fabric 데이터 저장소에 로드해야 하는 시나리오를 살펴보았습니다. 또한, Power BI 보고서 개발을 지원하기 위해 데이터 분석가가 데이터 소스에 더 가까운 위치에서 변환을 수행해야 하는 필요성도 파악했습니다.

Microsoft Fabric을 사용하면 Dataflows Gen2를 생성하여 레이크하우스의 데이터 통합을 수행하고, 선택적으로 데이터 파이프라인에 데이터 흐름을 포함할 수 있습니다. Dataflows Gen2에 대해 알아보고 데이터 통합 프로세스의 일부로 사용하는 방법을 살펴보았습니다. Power Query Online은 코드 작성 없이 복잡한 데이터 변환을 수행할 수 있는 시각적 인터페이스를 제공합니다.

데이터 통합에 대해 자세히 알아보려면 [Microsoft Fabric](https://learn.microsoft.com/en-us/fabric/data-factory/) 설명서의 Data Factory를 참조하세요.
